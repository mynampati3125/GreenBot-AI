🌱 GreenBot AI – Crop Disease Detection & Cure Recommendation


📌 Project Overview

GreenBot AI is an intelligent crop monitoring system that helps farmers detect crop diseases using image analysis and provides treatment recommendations.
The system analyzes uploaded crop images, predicts whether the crop is healthy or diseased, and generates a detailed report including confidence scores and treatment suggestions.

This project was developed as part of the Infosys Internship Program.

🚀 Features:

🔐 User Authentication (Login & Registration)

📷 Upload Crop Images for Analysis

🤖 AI-Based Disease Detection

📊 Prediction Confidence Visualization

🌿 Treatment & Care Recommendations

🧪 Soil & Fertilizer Guidance

📄 Downloadable AI Report (PDF)

💬 Integrated Chatbot for User Assistance

🧠 How the System Works

1.User logs into the application.
2.A crop image is uploaded through the interface.
3.The AI model analyzes the image.
4.The system predicts:
    Healthy crop
    Crop disease detected
5.The system displays:
    Confidence scores
    Graphical comparison
    Treatment & prevention steps
6.A detailed PDF report can be downloaded.

⚙️ Installation & Setup
1️⃣ Clone the Repository

2️⃣ Navigate to the Project
cd GreenBot-AI
3️⃣ Install Dependencies
pip install -r requirements.txt
4️⃣ Run the Application
streamlit run app.py

The application will open in your browser.
